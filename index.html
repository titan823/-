デザイン

トークン計算

※ オフ時は o200k_base による近似計算（≈95-98%）。オンにすると初回のみ約17MBのDeepSeek公式トークナイザーを取得し、100%正確なカウントを行います。

API による正確なトークン計算 (オプション)

※ オンにすると、入力停止時にバックグラウンドで各AIのサーバーに通信します。ほとんどのプロバイダではローカルトークナイザーで十分な精度が得られるため、通常は不要です。

※ DeepSeek には専用の token count API がないため、chat completion API で代替します。生成1トークン分の課金が発生します。

システム設定

※ オンにすると「保存」ボタンを押さなくても、変更が直ちに保存・適用されます。

※ 前半/後半入力欄・出力欄・検索/置換文字列などをブラウザに自動保存し、再読み込み時に復元します。

※ オフ: AIには正式名称で返答するよう指示します。
※ オン: AIには略号で返答するよう指示します（クライアント側で復元する前提）。

バージョン: 6.6 (セリフ結合・括弧置換・空白削除)

※ サーバー上の新しいコードを強制的に読み込みたい場合に使用します。

※ ブラウザに保存されている「圧縮辞書」「テーマ設定」「作業中テキスト」などを全て削除し、初期状態に戻します。

頻出ワード抽出結果
